# 一人公司 24h 搞钱执行清单

目标：24小时内拿到第一笔付费订单（哪怕9.9元），验证“流量 -> 咨询 -> 收款 -> 交付”闭环。

## 0. 先决条件（30分钟）

1. 准备收款码图片，文件名：`public/qr-pay.jpg`。
2. 确认 `.env.local` 里 AI key 可用。
3. 设置 Vercel Token（PowerShell）：

```powershell
$env:VERCEL_TOKEN="你的vercel_token"
```

## 1. 上线（30分钟）

1. 本地启动预览：

```powershell
cd "C:\Users\陈盈桦\Desktop\一人公司"
.\start.bat
```

2. 重点检查页面：
   1. `/`
   2. `/soul-autopsy`
   3. `/ai-fortune`
   4. `/pay?product=soul`
3. 执行部署：

```powershell
.\deploy.bat
```

## 2. 产品打单策略（6小时）

只卖一个低门槛爆品：`灵魂尸检报告（9.9）`。

1. 引流话术模板（短视频/小红书/朋友圈）：
   1. 标题：`把聊天记录丢进来，3分钟看透TA到底爱不爱你`
   2. 正文：`免费试读前30%，看完再决定要不要解锁完整版（9.9）`
   3. 行动：`私信我“尸检”领入口`
2. 首批内容发布量：
   1. 小红书 6 条
   2. 抖音 4 条
   3. 朋友圈 3 条
3. 私信成交动作：
   1. 先发免费体验链接
   2. 用户追问时只回答一句：`完整版会给你5条可执行动作，今天特价9.9`
   3. 立刻发支付页链接：`/pay?product=soul`

## 3. 第二增长点（6小时）

加售高客单：`爆文洗稿机（599）`，面向做内容变现的人群。

1. 发 3 条“前后对比”案例（原文 -> 改写后）。
2. 只接 3 个名额，制造稀缺：
   1. `今晚只收3位内测，交付1对1改稿策略`
3. 支付链接：`/pay?product=launderer`

## 4. 24小时节奏表

1. 第 1-2 小时：上线 + 跑通支付路径。
2. 第 3-8 小时：密集发内容 + 回私信。
3. 第 9-14 小时：复投高互动内容，继续引导到支付页。
4. 第 15-20 小时：推 599 内测名额。
5. 第 21-24 小时：复盘数据，保留高转化素材，第二天放量。

## 5. 最小复盘指标

1. 访问支付页人数（目标 100+）。
2. 点击“立即支付”人数（目标 20+）。
3. 成交单数（目标 3 单起）。
4. 客单结构（9.9 引流单 / 599 利润单）。

只要今天拿到第一笔，你的模型就成立了，第二天只做放量和提价。
